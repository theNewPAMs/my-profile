SciLab ABC

SciLab ABC is a lab that we build in my community using old computer. This lab uses old computers from companies and people which they were thinking of throwing away. To make sure that they are still efficent, we installed linux OS on them. The nice thing about linux dest is that is is openSource.

Beyond Class Room Project

This is a project that we started recently. Bacause of the lack of resources in small communities, which results in lack of skills amongst our youth who cant afford post-matric education, we have started a project where we teach the youth in small communities skills that we learn in varsity. This includes programming and how to use some of the technical softwares such as Revit.
